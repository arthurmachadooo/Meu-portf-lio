Meus Projetos

Projeto 1 - Jogo Da Velha

O Jogo da Velha é um clássico jogo de tabuleiro onde dois jogadores alternam turnos para colocar seus símbolos ("X" ou "O") em um grid 3x3, com o objetivo de formar uma linha de três símbolos consecutivos. Caso o tabuleiro se encha sem que alguém vença, o jogo termina em empate. No projeto de Jogo da Velha 2.0, os professores pediram que desenvolvêssemos uma versão digital do jogo com um timer para cada rodada, exigindo que os jogadores tomassem decisões rápidas. Essa versão não só aprimorou o clássico, mas também nos desafiou a aplicar conceitos de programação, controle de tempo e design de interfaces, enquanto trabalhávamos em equipe para criar uma experiência mais dinâmica e interativa.

Projeto 2 - Site De Presentes

O Projeto do Site de Presentes de Natal foi uma ideia criativa para surpreender minha namorada. Em vez de entregar os presentes de forma tradicional, criei um site onde ela tinha que completar pequenos desafios para descobrir o que havia ganhado. Cada desafio revelava um presente, tornando o momento mais interativo e divertido. Esse projeto me permitiu aplicar conceitos de design web e interatividade, criando uma experiência única e personalizada.
